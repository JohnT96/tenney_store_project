# Tenney Tech Store

## SDC310L Online Store Project

This project is a basic PHP and MySQL web application that simulates a simple online tech store.

## Features

- Product catalog page
- Database connection using MySQLi
- Organized folder structure:
  - /db (database export file)
  - /includes (database connection file)
  - /pages (catalog page)
  - index.php (home page)

## Technologies Used

- PHP
- MySQL
- XAMPP (Apache + MySQL)
- phpMyAdmin
- GitHub
